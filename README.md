# MONA - My Own Navigation Assistant

개인 감정, 루틴, 목표, 생각 등을 통합 관리할 수 있는 자기관리 웹 애플리케이션입니다.

---

## 🔧 실행 방법

1. 가상환경 활성화
   ```bash .\venv\Scripts\activate```
2. FastAPI 서버 실행
    ```uvicorn main:app --reload```
3. 웹 접속
 ```127.0.0.1:8000```
4. 🔐 테스트 계정
ID: test
PW: 1234

로그인 후 홈 화면에서 감정 기록, 루틴 관리 등 기능을 테스트할 수 있습니다
